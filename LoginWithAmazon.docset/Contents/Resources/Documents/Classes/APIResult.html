APIResult Class Reference

Inherits from NSObject
Declared in AIAuthenticationDelegate.h

Overview

This class encapsulates success information from an AIMobileLib API call.

  result

The result object returned from the API on success. The API result can be nil, an NSDictionary, or an NSString depending upon which API created the APIResult.

@property (retain) id result

Availability

1.0

Discussion

  • [AIMobileLib authorizeUserForScopes:delegate:] : Passes nil as the result to the delegate.
  • [AIMobileLib getAccessTokenForScopes:withOverrideParams:delegate:] : Passes an access token as an NSString object to the delegate.
  • [AIMobileLib clearAuthorizationState:] : Passes nil as the result to the delegate.
  • [AIMobileLib getProfile:] : Passes profile data in an NSDictionary object to the delegate. See the API description for information on the key:value pairs expected in profile dictionary.
